# VioletZineBot
Telegram Bot and web-panel for viewing student work.
Python, FastAPI and pyTelegramBotAPI.

### Installing dependencies:
Windows: 
```Shell
./init.bat
```

Linux:
```Shell
bash ./init.sh
```

### Run:
```Shell
python3 ./main.py
```

Recommendation: create a service to restart the bot together with the server.
